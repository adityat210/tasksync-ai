# TaskSync
TaskSync is intended to be a full-stack serverless collaborative task platform built to explore real-time multi-user workflows, structured task management, and practical AI features such as semantic search, detection for duplicates and summaries for tasks

## Goals for this project
- Support authenticated users, creating shared workspaces, boards, lists, tasks and comments
- Add real-time updates for task edits and moves
- Layer in useful features, attempt to ship project with clean commits and documentation

### Frontend stack
- Next.js
- TypeScript

### AWS Services
- Amazon S3
- Amazon CloudFront
- Amazon Cognito
- Amazon API Gateway
- AWS Lambda
- Amazon DynamoDB
- Amazon CloudWatch
